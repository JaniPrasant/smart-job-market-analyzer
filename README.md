# SMART JOB MARKET ANALYZER

**Project Overview**

This project analyzes real-time job market data using the Remotive Jobs API to identify hiring trends, most demanded skills, and experience-level distribution in the analytics and tech job market.

## Objectives

- Identify high-demand technical skills
- Analyze job title frequency
- Examine location-based hiring trends
- Classify experience-level distribution

### Data Source

Remotive Public Remote Jobs API

```javascript
url = "https://remotive.com/api/remote-jobs"
```

### Tools Used

```javascript
Python, Pandas, Requests, Matplotlib, Seaborn
```

## Key Insights

- Top job roles identified through frequency analysis
- Most demanded skills extracted using keyword aggregation
- Location trends analyzed using grouping techniques
- Experience-level distribution classified using rule-based logic

## Project Structure

- Notebook for data collection and analysis
- Saved cleaned dataset for reuse
- Visual outputs for insight presentation

## Business Value

This project demonstrates API integration, JSON handling, exploratory data analysis, and business-focused interpretation of hiring trends.
